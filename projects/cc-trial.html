Identifying Counterfeit Currency

A compact, practice-heavy module that helps frontline tellers spot counterfeit Indian currency with confidence.

Project Brief

The client needed an e-learning module for bank tellers to confidently detect counterfeit Indian currency.

Conducting Need Analysis
  • Common errors tellers make in detection
  • Denominations posing highest risk
  • Feasible time on task for tellers
  • Compliance requirements per RBI
  • Policies/processes to align
Defined Need

Design a 30–40 min engaging module covering all denominations, key security features and red flags, with realistic practice. Must be concise, highly interactive, and fit teller workflows.

Audience

Frontline bank tellers handling daily cash transactions.

Module Duration

~40–50 minutes (self-paced)

My Roles
Instructional Design Gamification Strategy Graphics & Mock-ups E-learning Development

Learning Design Strategy

The module was designed as a game-based experience inspired by a “heist” storyline, tightly interwoven with learning content.

  • Four heist levels—each focused on one denomination.
  • Explore metallic printing plates and security features in depth.
  • Production gets disrupted; some notes are corrupted.
  • Learners face knowledge-check challenges to spot counterfeit notes.
To bring this concept to life, I…
  • Designed the need analysis and game-aligned design document mapped to learning objectives.
  • Developed a step-by-step storyboard covering flow, interactivities, and knowledge checks.
  • Created visual mock-ups to align game mechanics with design elements.
  • Authored & published SCORM packages for seamless LMS deployment.

The Outcome

The client appreciated the practical relevance, clean design, and how easily tellers could engage and apply the skills on the job.
